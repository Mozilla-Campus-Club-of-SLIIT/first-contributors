# First contributors

> This project aims to provide a beginner-friendly platform for making a first open-source contribution.

## Quick Guide

1. Find an issue you can work on

2. Fork the repository  
   Click **Fork** on the top right of the repo.

3. Clone your fork  
```bash
git clone <your-fork-url>
```

4. Create a branch  
```bash
git checkout -b your-branch-name
```

5. Make your changes  
   Edit files as required.

6. Commit changes  
```bash
git add .
git commit -m "Your message"
```

7. Push to your fork  
```bash
git push origin your-branch-name`
```

8. Open a Pull Request  
   Go to the original repo → **Pull Requests** → **New Pull Request** → select your branch → submit.
